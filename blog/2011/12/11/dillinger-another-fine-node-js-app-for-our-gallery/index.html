Dillinger - Another Fine Node.js App for Our Gallery

Dec 11th, 2011

When discussing node clusters in the Nodejs group someone mentioned Dillinger, a cloud-enabled Markdown editor designed and developed by @joemccann. We immediately wanted this running on Cloudnode and now, here it is is:

Fig1: Dillinger Markdown Editor 

Visit http://dillinger.cloudno.de from your Mac, PC or iPad. The built-in connection to Github allows it to browse and edit the markdown files from all your repositories.
